World's Fastest Event Translations Are Here!

Saamer M•June 7, 2026

Get started with unimaginably fast captions & translations, impress yourself & everyone you know

Last Update 14 days ago

Step 1: After installing the ConferenceCaptioning app from here, and following the steps here to activate your Premium Stream account, choose the "Rapid" accuracy option.

Step 2: Once that is selected, tap on the "Start Premium Stream" button to go to the stream page. You should now see the "Open Translation page..." button. When you select your input & output languages, then tap on that Open button, it should open Google Chrome (if you have it installed) to the right page. Make sure the URL is http://localhost:8080/translation.html

Step 3: If you see the red bar on the top, you need to enable those flags on Chrome first if they are available and appear on the page. If that is done successfully and you reopen the browser, the red bar should disappear. If there are some flags that you aren't able to find in your browser, you can just ignore them since you might have a different version. Once the red bar is gone, the Start Translations button should activate and you can tap on it.

Step 4: Go back to the app and just tap on Start Live Streaming, and go back to the webpage in your browser. Scroll at the bottom to "Download All" the AI models needed for the language pair combinations, and all the other functionality should work perfectly too!

Notes: Make sure you don't have too many other apps open, you can ensure that less than 2GB of memory is being used in the Activity Monitor.

If you are facing a reduction in speed from increasing the characters to translate while translating into multiple languages simultaneously, you can also perform translations on a separate computer! In order to do that, you would need to trust your first computer as a safe origin from the second computer within chrome by going to this URL: chrome://flags/#unsafely-treat-insecure-origin-as-secure and adding the local IP address of the first computer like http://10.0.4.106:8080,http://10.0.4.106 and relaunching. You can get that value by tapping on the Info icon in the app when it is not live streaming.
