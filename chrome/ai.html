This application requires Chrome's built-in on-device AI. Follow these steps to enable it:

Step 1: You must use Google Chrome (preferably Dev or Canary builds) to use this feature.
Step 1: Copy and paste this into your URL bar:
chrome://flags/#optimization-guide-on-device-model
Set it to Enabled BypassPrefRequirement.
Step 2: Copy and paste this into your URL bar:
chrome://flags/#prompt-api-for-gemini-nano-multimodal-input
Set it to Enabled.
Step 3: Relaunch Chrome using the button at the bottom of the flags page.
Step 4: Go to chrome://components/
Find Optimization Guide On Device Model and click "Check for update". Wait for it to download.

Once completed, refresh this page. For more details, visit the Official Chrome AI Docs.
